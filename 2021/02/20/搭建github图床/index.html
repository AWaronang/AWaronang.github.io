搭建图床

使用PicGo+Github搭建图床

下载PIcGo

开始:

1.Github配置

打开GIthub

没有用户的话就自己创建一个用户

创建一个新的仓库

这里面的名字可以随便填写，仓库一定要为公开,然后点击创建即可

进入设置，再头像那里，点击就会出现这个，然后点击设置(Settings)

然后找到Developer settings

点击里面的Personal access tokens，然后再里面点击创建新的token

Note这里随便填写，下面的repo那个记得勾选,然后直接点创建

把这里的token复制下来，要记住，等下重新打开就没有了

2.PIcGo配置

第一个设置仓库名就是你的github的仓库名字

第二个设置你的分支名字设置master就好了

第三个设置Token就是你刚刚保存的

第四个就是存储路径
